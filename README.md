# COLLEGE_EXIT_FORM_DETECTION-JRU_Guidance-TripleKill
Sentiment Analysis and Topic Modeling for College Exit Forms

This project focuses on analyzing student feedback collected from college exit forms using natural language processing (NLP) techniques. The system applies sentiment analysis and topic modeling to extract meaningful insights from textual responses, helping institutions better understand student experiences, concerns, and overall satisfaction.

Built using Python and powered by Bidirectional Encoder Representations from Transformers via Hugging Face, the project classifies responses into sentiment categories and identifies key themes present in the data. Data processing and analysis are handled using Pandas, while visualizations are generated through Matplotlib. The entire workflow is executed in Google Colab for accessibility and ease of experimentation.

Key Features:

Automated sentiment classification of student feedback
Topic modeling to identify common themes and concerns
Data preprocessing and analysis using Pandas
Visualization of insights using Matplotlib
Cloud-based execution via Google Colab

Purpose:
This project aims to transform unstructured exit survey responses into actionable insights, enabling educational institutions to make data-driven improvements in academic programs, services, and student support systems.
